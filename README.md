# Value Betting Predictor

The aim of this project is to create an engineering perspective for sports betting. Instead of gambling on matches, this project can help you look at matches in a statistical and mathematical manner. However, note that this project does not guarantee winnings over a period of time; instead, it allows people to paper trade to see how a Deep Learning bot performs over time and develop its instincts and thinking process. People can benefit from this project to overcome their gambling habits by replacing them with a paper trading experience. 

Please also note that this project is designed strictly for educational purposes and does not provide financial advice, ROI guarantees, winning promises, or guaranteed model accuracy (such as always selecting the true favorite or underdog).

## Core Architecture & Interactive Learning

Unlike standard models that simply predict the winner, this system utilizes a custom PyTorch feed-forward neural network to calculate implied probabilities and compares them against live bookmaker lines to find valuable ROI opportunities.

* **Entity Embeddings:** Uses PyTorch `nn.Embedding` to map high-cardinality Team and League IDs into dense vector spaces, learning the specific "playstyle" of each club.
* **Continuous State Representation:** Embeddings are concatenated with 5-match rolling statistics (Points, Goal Difference, Motivation) to form a 52-dimensional input tensor.
* **Deep Processing & Stability:** The network utilizes two hidden layers (128 and 64 neurons) regulated by ReLU activations, `BatchNorm1d`, and `Dropout(0.5)` to prevent overfitting on smaller datasets.
* **Automated Interactive Learning:** A local server executes a nightly automated pipeline that settles previous wagers and performs a re-training cycle. This ensures the model iteratively updates its weights based on the latest match outcomes, learning from its successes and failures in real-time.

## Repository Structure

* `master.py`: The nightly automation orchestrator. It settles previous bets, fetches new API data, and triggers the online learning loop.
* `train.py`: Defines the PyTorch architecture and handles the training/validation loops.
* `config.py`: Securely loads environment variables and API keys via `python-dotenv`.
* `requirements.txt`: Python dependencies required to run the pipeline.

## Setup & Installation

**1. Clone the repository:**
`git clone https://github.com/aykan2004/DL_SportsBettingBot.git`
`cd DL_SportsBettingBot`

**2. Install Dependencies:**
`pip install -r requirements.txt`

**3. Configure Environment Variables:**
Create a `.env` file in the root directory and add your API keys:
`SPORTS_API_KEY=your_key_here`
`ODDS_API_KEY=your_key_here`
